# 📖 What are Bitwise Operators?

Bitwise Operators perform operations on the **binary representation** of numbers.

Instead of working on decimal values, they manipulate individual bits (0s and 1s).

---

# 📊 Binary Representation

Example

```java
int a = 5;
```

Binary representation

```text
Decimal : 5

Binary  : 00000101
```

Another example

```java
int b = 3;
```

Binary

```text
Decimal : 3

Binary  : 00000011
```

---

# 📖 Types of Bitwise Operators

| Operator | Name |
|----------|------|
| & | Bitwise AND |
| \| | Bitwise OR |
| ^ | Bitwise XOR |
| ~ | Bitwise NOT |

---

# 📖 Bitwise AND (&)

Returns **1** only if both bits are **1**.

Example

```java
int a = 5;
int b = 3;

System.out.println(a & b);
```

Binary Calculation

```text
  0101

& 0011

------

  0001
```

Output

```
1
```

---

# 📖 Bitwise OR (|)

Returns **1** if at least one bit is **1**.

Example

```java
System.out.println(5 | 3);
```

Calculation

```text
 0101

|0011

-----

 0111
```

Output

```
7
```

---

# 📖 Bitwise XOR (^)

Returns **1** only when bits are different.

Example

```java
System.out.println(5 ^ 3);
```

Calculation

```text
0101

0011

↓

0110
```

Output

```
6
```

---

# 📖 Bitwise NOT (~)

Flips all bits.

Example

```java
System.out.println(~5);
```

Calculation

```text
5

↓

00000101

↓

11111010
```

Output

```
-6
```

> 💡 **Note:** Java stores integers using **2's complement**, so `~5` evaluates to `-6`, not just the flipped binary value.

---

# 📖 Shift Operators

Shift operators move bits to the left or right.

| Operator | Meaning |
|----------|----------|
| << | Left Shift |
| >> | Signed Right Shift |
| >>> | Unsigned Right Shift |

---

# 📖 Left Shift (<<)

Moves bits towards the left.

Each left shift generally multiplies the number by **2**.

Example

```java
System.out.println(5 << 1);
```

Calculation

```text
5

00000101

↓

00001010
```

Output

```
10
```

Another Example

```java
System.out.println(5 << 2);
```

Output

```
20
```

---

# 📖 Right Shift (>>)

Moves bits towards the right.

Each right shift generally divides the number by **2**.

Example

```java
System.out.println(20 >> 1);
```

Calculation

```text
20

00010100

↓

00001010
```

Output

```
10
```

---

# 📖 Unsigned Right Shift (>>>)

Fills the leftmost bits with **0**, regardless of the sign.

Example

```java
int x = -8;

System.out.println(x >>> 1);
```

Useful in low-level programming and bit manipulation.

---

# 🧠 JVM Internal Working

Example

```java
int result = 5 << 2;
```

Execution Flow

```text
5

↓

Convert to Binary

↓

Shift Left

↓

Convert Back to Decimal

↓

20
```

# ⚠️ Common Mistakes

## ❌ Confusing Logical AND (`&&`) with Bitwise AND (`&`)

```java
if(a > 0 && b > 0)
```

Logical AND works on boolean expressions.

```java
5 & 3
```

Bitwise AND works on binary values.

---

## ❌ Confusing `>>` and `>>>`

`>>`

- Preserves the sign bit.
- Used for signed numbers.

`>>>`

- Fills leftmost bits with `0`.
- Ignores the sign bit.

---

## ❌ Forgetting Binary Conversion

Many students memorize outputs instead of understanding the binary calculation.

Always solve step-by-step.

---

# 💡 Best Practices

- Use arithmetic operators for normal calculations.
- Use bitwise operators only when binary manipulation is required.
- Use parentheses to improve readability in complex expressions.
- Understand binary representation before using shift operators.
- Avoid using bitwise operators in business logic unless necessary.
